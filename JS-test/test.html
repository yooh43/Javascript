問題01

header要素内にあるbutton要素をクリックしたらnav要素が表示され、もう一度button要素をクリックしたらnav要素が非表示されるようにJavaScriptで設定しなさい。

条件

  • header要素内にあるbutton要素にクラス名「close」がつくとバツ印になる。nav要素が表示されている時のみバツ印になるよう設定しなさい。
  • nav要素は非表示にしてある。nav要素にクラス名「open」がつくと表示される。

問題03

下記の入力欄に体重と身長を入力したらBMIが表示されるように設定しなさい。

条件

  • form要素及びinput要素に必要な属性は各自考えて設定する。
  • 計算式　体重(kg) / ( 身長(m) * 身長(m) )
  • id名「A03-bmi」にBMIを例のように表示する。
  • id名「A03-result」にBMIが30以上なら「肥満」、25〜29なら「肥満気味」、18.5～24.9なら「標準」、18.5未満なら「痩せすぎ」を例のように表示する。

体重と身長を入力してください

体重 kg
身長 cm

例）あなたのBMIは19.5で標準です。

問題04

条件

  • 1896年〜2021年までの期間で夏のオリンピックが開催された年だけ全て表示させる。

  • id名「A04-result」に例のように夏のオリンピックが開催された年を表示する
    表示するためのタグや属性は各自追加する。

夏のオリンピック　開催年

第2回　1900年
